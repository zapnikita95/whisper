# Как исправить права доступа на Mac

## Проблема: "Не удается запустить файл, поскольку у Вас нет соответствующих прав доступа"

### Решение 1: Через Терминал (самый простой)

1. Открой **Терминал** (Terminal.app)
2. Перейди в папку с файлом:
   ```bash
   cd ~/Desktop/whisper
   ```
3. Дай права на выполнение:
   ```bash
   chmod +x start-client-mac.command
   ```
4. Теперь двойной клик должен работать!

### Решение 2: Через Finder (графический способ)

1. Открой **Finder** и найди файл `start-client-mac.command`
2. Правый клик (или Ctrl+клик) → **"Свойства"** (Get Info)
3. В разделе **"Общий доступ и права доступа"** (Sharing & Permissions):
   - Убедись, что у тебя (твой пользователь) стоит **"Чтение и запись"** (Read & Write)
   - Если нужно, нажми замок 🔒 внизу, введи пароль
   - Измени права на **"Чтение и запись"**
4. Закрой окно свойств

**Но лучше использовать Терминал** — это надёжнее.

## Про странные даты (21 марта 2026)

Если видишь даты в будущем — это может быть:
- Проблема с часовым поясом системы
- Неправильное системное время
- Проблема с файловой системой

**Обычно это не критично** — файл всё равно будет работать после `chmod +x`.

Если хочешь исправить:
1. Системные настройки → Дата и время
2. Проверь часовой пояс и время
3. Или просто проигнорируй — на работу не влияет

## Быстрая проверка

После `chmod +x` проверь, что права установлены:
```bash
ls -l start-client-mac.command
```

Должно быть что-то вроде:
```
-rwxr-xr-x  1 nikitazaporohzets  staff  1901 Mar 21 00:35 start-client-mac.command
```

Видишь `x` в `-rwxr-xr-x`? Значит права на выполнение есть!
