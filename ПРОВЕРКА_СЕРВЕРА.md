# Проверка сервера на Windows

## Ошибка: "Connection reset by peer"

Если видишь эту ошибку на Mac, проверь:

### 1. Сервер запущен?

На Windows открой терминал и проверь:
```powershell
netstat -an | findstr :8000
```

Должно быть что-то вроде:
```
TCP    0.0.0.0:8000           0.0.0.0:0              LISTENING
```

### 2. Запусти сервер

```powershell
cd $env:USERPROFILE\OneDrive\Desktop\whisper
.\start-server.bat
```

Или напрямую:
```powershell
& "$env:USERPROFILE\.venvs\faster-whisper\Scripts\python.exe" .\whisper-server.py --host 0.0.0.0 --port 8000
```

### 3. Проверь доступность с Mac

В терминале Mac:
```bash
curl http://100.115.68.2:8000/
```

Должен вернуть JSON:
```json
{"status":"ok","model":"large-v3","device":"cuda","ready":true}
```

### 4. Брандмауэр Windows

Если не работает:
1. Открой "Брандмауэр Защитника Windows"
2. "Дополнительные параметры"
3. "Правила для входящих подключений" → "Создать правило"
4. Тип: "Для порта" → TCP → 8000
5. Действие: "Разрешить подключение"
6. Применить ко всем профилям

Или временно отключи брандмауэр для теста.

### 5. Tailscale

Проверь, что оба устройства в одной Tailscale сети:
- Windows: `tailscale status`
- Mac: `tailscale status`

IP должны быть видны друг другу.

### 6. Логи сервера

Смотри в окне сервера на Windows — там должны быть сообщения о подключениях.
